# Select
Selects are used to provide a list of options for the user to choose from.\
They are used when the user needs to select one or more options from a list of options.\
They are used in forms, dialogs, and menus.

## Variants:
A well-chosen variant makes your design radiant.\
Choose the one that fits your design system the best.\
Default is Contained
